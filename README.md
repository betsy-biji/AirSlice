# 🍉 AirSlice – Gesture-Based Fruit Slicing Game

AirSlice is a computer vision-based game inspired by Fruit Ninja. Players use real-time hand gestures via webcam to slice falling fruits in the air.

## ✨ Features
- ✋ Hand tracking using MediaPipe  
- ✂️ Gesture-based slicing  
- 🍎 Multiple fruits + 💣 bombs  
- 🌍 Level-based backgrounds  
- 💥 Splash effects  
- 🔁 Restart system  

## ⚙️ Tech Stack
- Python  
- OpenCV  
- MediaPipe  
- NumPy  

## ▶️ How to Run
```bash
pip install opencv-python mediapipe numpy
python main.py
